# EDUSTART
Welcome to EDUSTART, a modern and responsive educational platform website. This project is built using HTML, CSS, and JavaScript (with GSAP animations) to create an interactive and engaging user experience. The website provides information about courses, services, and contact details, while incorporating a clean design and animations triggered by scroll events.
# Features
1. Responsive Design: The website is mobile-friendly and adjusts its layout for different screen sizes using media queries.
2. GSAP Animations: Smooth animations are added with the GSAP library and ScrollTrigger to enhance the user experience.
3. Hamburger Menu: A responsive navigation menu that collapses into a hamburger icon on smaller screens.
4. Course Cards: Highlighting key courses like Web Development, App Development, and Graphic Designing.
5. Scroll-Based Animations: Text, images, and cards appear with smooth transitions as users scroll through the page.
# Technologies Used
1. HTML5: Structure of the website.
2. CSS3: Styling for the layout, responsiveness, and hover effects.
3. JavaScript: Interactive functionality for the hamburger menu.
4. GSAP: Used for animations triggered by user scrolling.
5. Google Fonts: The "Poppins" font is used for typography.
